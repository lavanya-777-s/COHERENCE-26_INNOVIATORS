# ArthSetu – Budget Flow Intelligence & Leakage Detection Platform

## About
- AI-powered platform for tracking the flow of public funds.
- Detects financial anomalies and inefficiencies in budget utilization.
- Helps improve transparency and data-driven governance.

## Features
- Budget allocation vs spending analysis
- AI-based anomaly detection
- Detection of overspending and underutilization
- Financial insights dashboard
- Fund lapse prediction
- Transparency layer

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- AI/Analytics: Python, Pandas, Scikit-learn
- Database: MongoDB

## Goal
- Enable transparent and efficient public financial governance.
